Introduction

Welcome to the SmoothMath documentation. SmoothMath is a Python package designed to simplify mathematical operations and provide a smooth experience for developers.

Installation

To install SmoothMath, use the following command:

pip install smoothmath

Usage

Here's how you can use SmoothMath in your projects:

import smoothmath as sm

# Example usage
result = sm.add(5, 3)
print(result)

API Reference

Detailed API reference for SmoothMath functions:

add(a, b)

Adds two numbers and returns the result.

sm.add(a, b)
